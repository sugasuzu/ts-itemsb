### あなたの本来の研究目的（理解を確認したい）

**「0付近ではなく0から離れた小集団を発見する」**

これは以下のどちらを意味していますか？

#### パターンA: 両側の極端な値
```
目的: データ分布の中心(μ≈0)から離れた、極端な値を持つ小集団

例:
- 強い上昇パターン: mean >> 0 (例: +1.0%, +2.0%)
- 強い下落パターン: mean << 0 (例: -1.0%, -2.0%)

つまり: |mean - μ| が大きい
```

#### パターンB: 分散が極端に小さい/大きい
```
目的: 通常のランダム変動(σ=0.5%)と異なる分散特性を持つ小集団

例:
- 極低分散: sigma << 0.5 (予測可能性が高い)
- 極高分散: sigma >> 0.5 (ボラティリティが高い)
```

#### パターンC: 条件付き分布の発見
```
目的: 特定条件下で分布が大きく変化する瞬間を発見

例:
- 通常時: μ=0.0077%, σ=0.51%
- ルール発火時: μ=1.5%, σ=0.3% (分布が大きくシフト)

つまり: 市場構造が変わる「レジームチェンジ」を検出
```

#### パターンD: 多変量での外れ値
```
目的: (mean, sigma) の2次元空間で通常と異なる領域にあるパターン

例:
- 高mean & 低sigma: 強く安定した上昇
- 高mean & 高sigma: 強いがリスクも高い
- 低mean & 低sigma: 安定しているが期待値も低い
```

## 「小集団」の意味

「小集団を発見する」とは:
- サポート数が少ない（希少なパターン）
- データ全体では稀にしか起こらない
- しかし起こった時には明確な特性を持つ

という理解で正しいですか？

## 統計的な意味での「外れ値」

データ全体から見て統計的に異常な値を持つパターンを発見する、
という理解で正しいですか？

例:
- データ全体: μ=0.0077%, σ=0.51%
- 発見したいルール: μ=1.5% (約3σ離れている)
- これは「0から離れた」であり「データ平均から離れた」
